## BeatSafe

An application which allows users to upload their own samples, tag samples, share beats/sample packs, and download  packs or beats.

## Tech Stack
- Frontend: React, Tailwind
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Deployment: TBD

## Features
- User authentication
- CRUD operations
- Search/filter
- File uploads
- API integration
- Responsive UI
